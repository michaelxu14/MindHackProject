Doomed to Scroll is a brain visualizer webapp coded using JavaScript and using 3JS. 
Styling has been done using CSS.
We have integrated a 3d model of the human brain and have created a visual representation of how specific activities affect the brain.
More information can be found in the More Info tab

Users can select specific behaviors and activities(e.g., scrolling, gambling, exercising) to see simulated effects on the brain.
Clicking on a behavior displays additional information explaining how that activity may impact cognitive processes, attention, and mental well-being.
Visual simulations illustrate how different activities stimulate or affect areas of the brain.
